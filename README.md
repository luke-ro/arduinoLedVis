# arduinoLedVis
Controls a Ws2812b led strip with arduino Uno to visualize music. 

[Here](https://drive.google.com/file/d/1UZP99OhYk3BDIzZ4B_vjEh-FgU5lMlQU/view?usp=sharing) is a sample of the LED strip in action.
